# ZentroX — Video Automation Platform

## Project Overview
Platform for automated short-form video content creation and publishing (TikTok, YouTube Shorts). Russian-language UI.

## Technical Stack
- Next.js 14 (App Router), TypeScript, Tailwind CSS, Prisma, PostgreSQL
- NextAuth for auth (email/password + Google SSO placeholder)
- AWS S3 for file storage, Abacus AI LLM API for generation, Abacus AI FFmpeg API for video processing

## Design
- Dark theme: `--primary: 263 60% 58%`, `--background: 240 6% 7%`
- Badge workaround: Use plain `<span>` with badge-like classes
- All animations use position-only transforms (no opacity in keyframes)

## Architecture
- Video gen is async: POST returns videoId, background generates images→S3→FFmpeg→concat. Client polls 3s.
- LLM images are base64 → must upload to S3 first for FFmpeg
- Progress in `GeneratedVideo.generationLog` JSON: `{ step, progress, currentScene, totalScenes }`
- Publishing: fire-and-forget simulation with 95% success rate
- Trends fetched from DB and injected into script generation prompts
