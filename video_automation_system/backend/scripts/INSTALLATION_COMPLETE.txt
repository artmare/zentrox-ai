🚀 БЫСТРЫЙ СТАРТ:

1. Автоматическая установка:
   cd /home/ubuntu/video_automation_system/backend/scripts/
   ./setup.sh

2. Или ручная настройка:
   pip install -r requirements.txt
   cp .env.example .env
   nano .env  # Добавьте API ключи
   psql -U postgres -f database_setup.sql

3. Первый запуск:
   python example_usage.py

🔑 ТРЕБУЕМЫЕ API КЛЮЧИ:

1. YouTube Data API v3:
   • Получить: https://console.cloud.google.com/apis/credentials
   • Квота: 10,000 единиц/день (бесплатно)

2. Abacus AI API:
   • Ключ автоматически подтягивается из окружения
   • Переменная: ABACUS_API_KEY

3. PostgreSQL Database:
   • Установить: sudo apt-get install postgresql
   • Создать БД: database_setup.sql
